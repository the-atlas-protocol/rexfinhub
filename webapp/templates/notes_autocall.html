Autocallable Note Simulator

Backtest autocallable notes against historical reference data. Single product or basket of up to 5 references.

About this tool

What it does

For any autocallable note specification (references, barriers, tenor, memory) and any historical issue date, this tool replays what the actual cashflows would have been against historical price data. It then prices the “par” coupon — the rate that makes the note worth its issue price under risk-neutral assumptions.

How the suggested coupon is computed

We run a 10,000-path Black–Scholes Monte Carlo simulation of the references under risk-neutral GBM dynamics, replay the autocallable cashflows on each path, and bisect on the coupon rate to find the par-pricing coupon (PV = 100). Inputs:

  • Volatility: 252-day realized log-return stdev of each reference, ending on the issue date.
  • Correlation: Pearson correlation matrix from the same return panel (Cholesky-factored, PSD-repaired if needed).
  • Risk-free rate: 4% constant (proxy for indicative pricing).
  • Dividend yield: 0% for total-return indices, 2% for SPX/NDX price-return.
  • Funding spread: 0 (issuer-specific in reality; indicative pricing only).

During fast scrub interactions, a vol-based heuristic shows an instant estimate; the BS coupon overwrites within ~1s after you stop.

What this is NOT

This is not a real product price. Real autocallables are priced by issuers using calibrated vol surfaces, term-structure of rates, funding curves, and bid-ask. Treat suggested coupons as directional indicative values, not quotes. Historical-Outcomes sweeps use the faster vol heuristic for speed.

Data

Daily index closes for 26 references (SPX, NDX, BMAXUS, MerQube vol-targeted, BBG bond TR, etc.). Source: Bloomberg, exported manually, refreshed monthly. Pre-inception cells are skipped via LOCF.
